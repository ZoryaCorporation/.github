## About

Zorya Corporation develops **industrial-grade systems software** with a focus on:

- **Compiler Engineering** — Production language implementations
- **Runtime Systems** — GPU compute, AI inference, memory management
- **Legacy Modernization** — Liberating COBOL from vendor lock-in
- **Safety-Critical Code** — ZORYA-C standard for auditable C

We build tools that **show their work** — fully documented, fully traceable, fully open.
