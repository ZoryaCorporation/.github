## About

Zorya Corporation develops **industrial-grade systems software, tools and applications** with a focus on:

- **Compiler Engineering** — Production language implementations
- **Runtime Systems** — Performance focused modular frameworks adapted for modern development standards 
- **Affordability** — From user facing products to free tools, quality standards are maintained   
- **Safety-Critical Code** — ZORYA-C standard for auditable C

We build tools that **show their work** — fully documented, fully traceable, fully open.
